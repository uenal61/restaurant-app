# restaurant-app

Dieses Projekt ist eine moderne Webanwendung basierend auf:

- **React** mit **Vite**
- **TypeScript**
- **TailwindCSS**
- **Firebase** (für Authentifizierung & Firestore)

Ziel: Ein Bewertungssystem für Restaurants im Rahmen des „FridaysDining“-Projekts.
